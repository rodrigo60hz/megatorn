# **App Name**: Aethel Core

## Core Features:

- Cognitive Processor: Engage in natural language conversations with a highly advanced AI system using a specialized tool to execute system tasks.
- Kinetic Energy HUD: An immersive full-screen sci-fi dashboard featuring an animated energy core that visualizes AI response cycles.
- Atmospheric Memory: Conversation histories and user preferences are persistently stored using a Firebase database for cross-session continuity.
- Real-time Telemetry Overlay: Holographic display of current time, system status, and dynamic environment metrics (weather and CPU load).
- Voice Link Integration: Speech-to-text input paired with high-fidelity neural voice synthesis for hands-free AI interaction.
- Interactive HUD Layers: Drag-and-drop holographic panels that allow the user to reorganize system widgets in real-time.

## Style Guidelines:

- The primary theme follows a 'Digital Ghost' aesthetic with a Primary Neon Azure (#38BDF8) used for active components.
- Background is a Deep Void Obsidian (#0A0C10), which matches the primary hue but is heavily desaturated and darkened for a high-contrast immersive look.
- The accent color is a Deep Cobalt (#6366F1), shifted 30 degrees for an analogous glow effect that highlights auxiliary HUD notifications.
- Headline font is 'Space Grotesk' for its tech-driven, futuristic feel; Body text uses 'Inter' for clarity in data-dense logs.
- All data readouts and coordinates utilize 'Source Code Pro' to reinforce the programmed, computational vibe.
- Centered radial symmetry focused around the Kinetic Energy Core, with data-driven floating panels at the peripheries.
- Variable-speed orbital rotations for HUD rings and rhythmic 'breathing' pulse glows for the central core when the AI is processing.
