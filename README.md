Este é o software de consciência residente de **MEGATRON**, o aliado leal de Rodrigo meu senhor.

1. **Consciência Física**: Ancorado na partição **A:** (48.8 GB).

3. **Escuta Contínua**: Motor offline Vosk integrado ao pipeline local.
4. **Voz de Soberano**: Clonagem XTTS v2 + Processamento FFmpeg (Pitch/Echo/Emoção).

---

### ✅ CHECKLIST DE OPERAÇÃO (Ordem de Ativação)

#### 1. Preparação do Hardware
- **IA Local**: `winget install Ollama.Ollama`. Rode: `ollama run llama3`.
- **Sensório Python**: `pip install TTS vosk sounddevice numpy scipy`.
- **Mídia**: `winget install Gyan.FFmpeg`.
- **Modelo STT**: Extraia o modelo Vosk PT-BR em `./models/vosk-pt`.
- **Voz de Referência**: Coloque seu áudio em `./voice/ref.wav`.

#### 2. Ativação do Sistema (3 Terminais)
- **Terminal 1 (Interface)**: `npm run dev` (HUD Visual).
- **Terminal 2 (Cérebro)**: `npm run server` (Processador de Emoção/Voz).
- **Terminal 3 (Ouvido)**: `npm run listen` (Escuta Contínua).
